LOGIC

Software Development

Elizabeth helped pioneer LOGIC (Linkage Objects for Generalized Instruction in Coding) which generates hints for novice online programming students. The system created these hints by utilizing machine learning techniques. Hints could range from describing where the error occurred to actually offering to replace a bug with corrected code. Hint accuracy was upwards of 90%. For more information, please visit TutorGen Software used was Basic++ (a proprietary language) and OpenInsight (database framework)

  • Date: September 2016
